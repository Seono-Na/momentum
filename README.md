# momentum

## This is a "To Do list" webservice

Web service like ChromeApp "Momentum"

### Fuction

- To Do List
  - Add list
  - Done
  - Delete
  - List clear
  - Remember user and to do list
- Clock
- Geocording API
  - Location
  - Weather
  - Temperatures
- Linkbox
- Quotes
  - Renew every 15 seconds
- Change Background

### Caution

When logging out, all lists are cleared.

This service will not be responsible for that.
